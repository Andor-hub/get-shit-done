**HabitX/HabitXWidget/Providers/SmallWidgetProvider.swift:**
- `import WidgetKit`, `import SwiftUI`, `import SwiftData`, `import AppIntents`
- `struct SmallWidgetProvider: AppIntentTimelineProvider` with:
  - `typealias Intent = HabitWidgetIntent`
  - `typealias Entry = HabitWidgetEntry`
  - `func placeholder(in context: Context) -> HabitWidgetEntry`: return `HabitWidgetEntry(date: .now, habitSnapshot: nil)`
  - `func snapshot(for configuration: HabitWidgetIntent, in context: Context) async -> HabitWidgetEntry`: call `await makeEntry(for: configuration.habit)`
  - `func timeline(for configuration: HabitWidgetIntent, in context: Context) async -> Timeline<HabitWidgetEntry>`:
    1. `let entry = await makeEntry(for: configuration.habit)`
    2. Calculate midnight: `let midnight = Calendar.current.startOfDay(for: Calendar.current.date(byAdding: .day, value: 1, to: .now)!)`
    3. Return `Timeline(entries: [entry], policy: .after(midnight))` — refresh at midnight for daily habit reset
  - `@MainActor private func makeEntry(for entity: HabitEntity?) -> HabitWidgetEntry`:
    1. Guard unwrap entity, return nil-snapshot entry if nil
    2. `let context = SharedModelContainer.container.mainContext`
    3. `let id = entity.id`
    4. `let descriptor = FetchDescriptor<HabitSchemaV1.Habit>(predicate: #Predicate { $0.id == id })`
    5. Guard unwrap `try? context.fetch(descriptor).first`
    6. Return `HabitWidgetEntry(date: .now, habitSnapshot: HabitSnapshot(habit: habit))`
- CRITICAL: `makeEntry` must be `@MainActor` — `mainContext` is MainActor-isolated
- CRITICAL: Use `FetchDescriptor`, NOT `@Query` — `@Query` is unavailable in TimelineProvider

  - `func getSnapshot(in context: Context, completion: @escaping (MediumWidgetEntry) -> Void)`: wrap in `Task { @MainActor in completion(makeEntry()) }`
  - `func getTimeline(in context: Context, completion: @escaping (Timeline<MediumWidgetEntry>) -> Void)`: wrap in `Task { @MainActor in ... }`, call makeEntry(), compute midnight, return `Timeline(entries: [entry], policy: .after(midnight))`

    1. `let context = SharedModelContainer.container.mainContext`

    5. Return `MediumWidgetEntry(date: .now, snapshots: snapshots)`
- CRITICAL: Callback-based provider needs `Task { @MainActor in }` wrapping for Swift 6 concurrency
