Each row: habit name (.headline) and below it a quick streak label like "Current streak: X days" in .subheadline .secondary, computed via StatsCalculator.currentStreak(for:).

    2. **HabitStatsView.swift**: Takes habit: HabitSchemaV1.Habit. Navigation title: habit.name. Layout as a ScrollView with VStack:

    **Stats Cards Section** — Three stat cards in a horizontal layout (LazyVGrid with 3 columns or HStack):
    - Card 1: "Current Streak" label, "\(StatsCalculator.currentStreak(for: habit))" large number, "days" subtitle. Use accent color for the number.
    - Card 2: "Best Streak" label, "\(StatsCalculator.bestStreak(for: habit))" large number, "days" subtitle. Use accent color.
    - Card 3: "30-Day Rate" label, "\(Int(StatsCalculator.completionRate30Days(for: habit) * 100))%" large number. Use accent color.
    Each card: RoundedRectangle background with Color(.secondarySystemBackground), padding, centered text.

    ```swift
    Chart(completionData, id: \.date) { entry in
        BarMark(
            x: .value("Date", entry.date, unit: .day),
            y: .value("Completed", entry.completed ? 1 : 0)
        )
        .foregroundStyle(entry.completed ? HabitDefaults.appAccentColor : Color(.systemGray4))
    }
    ```
    Chart height: ~200pt. X-axis: show abbreviated day labels. Y-axis: hidden (binary 0/1).

    CRITICAL (per STAT-04): Missed days use `Color(.systemGray4)` — a neutral, light gray. Do NOT use red, orange, yellow, or any warning/error color. No "missed" or "failed" labels. The bar is simply shorter/grayed. The chart title should be "Last 30 Days" — neutral framing.

    Below the chart, optionally add a small note: "X of 30 days completed" in secondary text.

    Use HabitDefaults.appAccentColor consistently for positive/completed states (per D-12).
